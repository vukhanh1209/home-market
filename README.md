# FRESH FOOD E-COMMERCE WEBSITE
## Use Case:
**Customer:**
- Search Product
- Sign Up

**Registered customer:**
- View Cart
- Manage Cart
- Log In
- Log Out
- Check Out
- Place Order
- Cancel Order
- Check Order Status
## Technologies:
**Front-end: HTMLL, CSS, Tailwind, JavaScript, ReactJS**

**Back-end: Java, Spring Boot**

**Database: MySQL**
